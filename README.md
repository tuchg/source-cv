# SourceResume

## Development

### setup

Firstly, we should create a `.env` file in the root directory of project repository and write the following content into
it.

```
OPENAI_API_BASE="https://api.openai.com/v1"
OPENAI_API_KEY="sk-XXXX"
```

### run

```bash
pnpm install
pnpm run dev
```
